- 🧠 **Problem-Solving Process**

- ✍️ **Final Year Report Preparation**  
  As I begin writing my report, this repository will also be used to:

Separating repositories allows for:
- Clear distinction between **team development work** and **individual learning/process tracking**
- Better organisation of **technical challenges and solutions**
- A structured way to manage both **development and academic reporting**
